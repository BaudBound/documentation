# Runner Updates

Desktop builds check the signed update feed at startup. When a newer semantic version is available, the UI presents release information and lets the user decide whether to download it.

Download progress is shown in the update dialog. The application verifies the downloaded artifact before installation.

After a verified download, choose Restart to install and launch the new version.

Headless operators and Linux users whose installation cannot be replaced automatically should download the new release from the project's GitHub releases. Never bypass signature verification.
