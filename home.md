# BaudBound

BaudBound is a visual automation platform for building workflows as connected nodes and running them on Windows or Linux.

Use the [public editor](https://editor.baudbound.app/) to create and export `.bbs` script packages. Install the BaudBound runner on the machine that should execute those packages.
